          Start Launcher
            Nova --launch
          Start New Game wizard
            Nova --new
          Start Race Designer
            Nova --race
          Start Component Editor
            Nova --components
          Start Console
            Nova --console
          Start GUI
            Nova --gui -r <race> -t <turn> -i <intel file> -s <state file>
          Run AI
            Nova --ai -r <race> -t <turn> -i <intel file>
          Display the help screen
            Nova --help
